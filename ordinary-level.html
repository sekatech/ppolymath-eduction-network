Instructions for Adding New Subjects

To add a new subject:

  1. Copy the subject card template (marked in the code with comments)
  2. Replace "subject-name" with the actual subject name (use lowercase with hyphens for spaces)
  3. Replace "Subject Name" with the display name
  4. Choose an appropriate icon from Font Awesome (fas fa-icon-name)
  5. Update the data-subject attribute to match the subject name
  6. Update all ID attributes to match the subject (e.g., 'subjectQuestions', 'subjectSolutions')
  7. Add the subject to the subjects object in the JavaScript generateYearGroups() function
  8. Add contact info to the footer

Example: For "Religious Studies", use data-subject="religious studies" and icon <i class="fas fa-pray"></i>
